Kinect2Scratch4Mac

The bridge application to connect "KINECT" to Scratch on Mac OSX

About

This is Mac version of an application called Kinect2Scratch, which is developed by Stephen Howell. It sends data from Microsoft Kinect controller to Scratch, which is the programming environment developed by MIT Media Laboratory.

Anyway...
You can use Kinect from Scratch on Mac!!!!

Video

How To Use

Required

  • Kinect
  • Mac OSX(10.6.8~)

    • I don't check whether run this app on 10.5 or not.

  • Xcode(4.x)

    • You can get it from here.

Build

  1. Download a package of Kinect2Scratch4Mac(openFrameworks007 included) from here(not ready)
  2. Unpack it, and open "Kinect2Scratch.xcodeproj"
  3. Change a scheme of your mac from the upper left select box
  4. Run it(press command-r)

    5. A application builded is at 'apps/myapps/kinect2Scratch4Mac/bin'.

    ! Don't move an app from bin folder. Please make alias from where it is. If relation of place is changed, the app doesn't run.

Using Windows?
